# 📖 BOOKSTORE CMS <a name="about-project"></a>


**bookstore CMS** is a simple project to manage a books inventory, created in React and uses Redux to manage state.

### Tech Stack <a name="tech-stack"></a>

## 💻 Getting Started <a name="getting-started"></a>
To get a local copy up and running, follow these steps.
1. Clone this repository
   run 
   **git clone git@github.com:marvin-nyalik/bookstore-redux.git**

2. Open the project in a text editor
3. Run `npm install` to install dependencies
4. Run `npm start` to run the project 
5. Interact with project in your favorite browser

### Prerequisites

In order to run this project you need:

1. Install git on your local computer
> Find git installation details here https://git-scm.com/book/en/v2/Getting-Started-Installing-Git

2. Clone this repository
> Run this command '**git clone git@github.com:marvin-nyalik/bookstore-redux.git**'
